### Interactive Component

You can embed React components inside your post:

```jsx
const Counter = () => {
  const [count, setCount] = React.useState(0);
  return (
    <div>
      <p>Counter: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increase</button>
    </div>
  );
};

<Counter />;
```
